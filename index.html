Part I

Foundations: P-values and Error Rates

Every method in this course rests on one idea: a valid p-value satisfies P(p ≤ t) ≤ t for all t ∈ [0,1] under the null — the super-uniform property. When many hypotheses are tested simultaneously, controlling the rate of false positives requires understanding how p-values behave collectively.

FWER — Family-Wise Error Rate

FWER = P(V ≥ 1)
= P(at least one false rejection)

The strictest criterion: control the probability that any true null is falsely rejected. Appropriate when even a single false positive is costly — e.g., clinical trials with regulatory consequences. Bonferroni and Holm control FWER.

FDR — False Discovery Rate

FDR = E[V / max(R, 1)]
= expected proportion of false discoveries

A less conservative criterion that permits some false positives as long as they constitute a controlled fraction of all discoveries. Standard in genomics, neuroimaging, and other high-throughput sciences. Benjamini–Hochberg controls FDR.

Key distinction: When all nulls are true (m₀ = m), FDR reduces to FWER, because if any rejection occurs, the FDP is exactly 1. FDR is strictly less conservative than FWER only when some nulls are actually false.
Part II

Testing the Global Null

The first question in any multiple testing problem is: is there any signal at all? A global null test rejects H₀ = ∩ H₀,ᵢ — the hypothesis that every individual null is true — at level α. Three classical tests differ in their dependence assumptions and power profiles.

A warning from Lecture 2: If you compute both Bonferroni and Fisher and report whichever one rejects, the combined Type I error can approach 2α. Fix: run each at α/2 and reject if either rejects. This is itself a multiple-testing correction — applied to the procedures rather than the hypotheses.
Part III

Multiple Testing Procedures

Moving from "is there any signal?" to "which hypotheses are non-null?" requires a procedure that maps p-values to a discovery set while controlling error. The two workhorses are Holm (FWER) and Benjamini–Hochberg (FDR). They differ fundamentally: Holm is step-down (stop at the first failure); BH is step-up (continue past failures as long as the largest threshold is met).

From Lectures 3–4: BH is a step-up procedure — "the more you reject, the more you can allow yourself to continue rejecting." Its threshold αk/m increases with k. Holm is step-down: thresholds α/(m+1−k) decrease, and a single failure to cross a threshold stops all further rejections. BH controls FDR ≤ α·m₀/m under independence; Holm controls FWER ≤ α under arbitrary dependence. When all nulls are true, BH reduces to the Simes global test.

Part IV

Adaptive BH & Estimating π₀

Standard BH controls FDR at α·m₀/m, not at α. When many nulls are false (π₀ = m₀/m is small), BH is conservative and wastes power. The Storey estimator π̂₀ uses the flat right tail of the p-value histogram to estimate π₀, then inflates BH's threshold to α/π̂₀.

Storey's π₀ Estimator

Choose a cutoff γ. P-values above γ are mostly from true nulls (which are uniform), so π̂₀(γ) = #{pᵢ > γ} / [m(1−γ)] estimates the null proportion. The dashed line shows π̂₀ as a function of γ on the current data.

The +1 correction — π̂₀(γ) = (1 + #{pᵢ > γ}) / [m(1−γ)] — prevents π̂₀ from being zero, which would make α/π̂₀ blow up. A data-splitting variant uses p-values above γ only for estimation and p-values below γ only for rejection, restoring rigorous finite-sample FDR control.
Part V

Permutation & Randomization Tests

When the null distribution of a test statistic is analytically intractable, permutation tests construct it empirically by exploiting exchangeability. Under the null, the observed statistic is just one of many equally likely values, and the permutation p-value is the rank of the observed statistic among its permuted counterparts.

We test whether X and Y are independent by computing a test statistic (correlation) on the original pairing, then repeatedly permuting the X-labels. The histogram shows the null distribution built from permutations, with the observed statistic marked.

From Lectures 7–8: The permutation p-value is p = (1 + #{T̃ᵦ ≥ T}) / (1 + M). The +1 in numerator and denominator ensures p is never zero and is always super-uniform under the null. Three canonical settings are covered: (1) both marginals known → simulate from the product distribution, (2) one marginal known → resample that variable, (3) neither known → permute labels. Conditional independence testing (H₀: X ⊥ Y | Z) is harder — local permutations within Z-clusters trade validity against power, and the curse of dimensionality makes this fundamentally difficult when Z is high-dimensional.
Part VI

Selective Inference & Knockoffs

When you choose which hypothesis to test after looking at the data — by selecting a model via Lasso, or by picking the largest estimated effect — classical p-values and confidence intervals are invalid. Selective inference conditions on the selection event; knockoffs use fake variables as negative controls.

Post-Selection Inference

Lectures 10–11
Conditional Coverage

The Lasso's KKT conditions imply that the event {selected model = (S, sign)} is a polyhedron {y : Ay ≤ b} in observation space. Conditional on this event, the distribution of any linear functional η⊤Y is a truncated Gaussian, from which valid p-values and confidence intervals can be derived.

Three correction strategies: (1) Data splitting — select on one half, infer on the other. Always valid, but wastes data. (2) Multiplicity correction — Bonferroni over all possible selections. Often too conservative. (3) Selective inference — condition on the polyhedral selection event. Exact, but computationally more involved.

Model-X Knockoffs

Lectures 14–15
FDR Control

Construct fake copies X̃₁,…,X̃ₚ of the covariates that are pairwise exchangeable with the originals: swapping Xⱼ and X̃ⱼ doesn't change the joint distribution. Null variables and their knockoffs are indistinguishable, so selected knockoffs serve as negative controls for estimating FDP.

FDP̂(S) = #{j : X̃ⱼ ∈ S} / max(1, #{j : Xⱼ ∈ S})

For Gaussian X ~ N(μ, Σ), the knockoff pair has covariance [Σ, Σ−D; Σ−D, Σ] for a diagonal D ≥ 0 making the matrix PSD. Conditional on max(Zⱼ, Z̃ⱼ), each null feature's "coin flip" — whether Xⱼ or X̃ⱼ has the larger importance — is an independent Bernoulli(1/2). This yields FDR control without independence of the test statistics.

Part VII

Conformal Prediction

Conformal prediction constructs prediction intervals with distribution-free, finite-sample coverage guarantees. The only assumption is exchangeability of the data — no parametric model, no smoothness, no consistency of the underlying predictor. The quality of the model affects interval width, never validity.

Split Conformal Prediction — Interactive

Data is generated from Y = f(X) + noise. A regression model is fit on the training set, and conformity scores |Yᵢ − f̂(Xᵢ)| are computed on the calibration set. The prediction interval at each test point is f̂(x) ± q̂, where q̂ is the (1−α)(1+1/n)-quantile of the calibration scores. Adjust model quality and noise level to see how intervals respond.

The unifying principle across all methods: Validity comes from respecting the relationship between what you looked at and what you claim. Bonferroni pays for looking at m hypotheses. BH pays proportionally to how many you reject. Selective inference pays for the specific model you chose. Conformal prediction pays nothing for model choice — but only guarantees marginal, not conditional, coverage. Every method in this course is a different way of accounting for the information the data has already revealed.
